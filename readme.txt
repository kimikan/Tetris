
Description:
   This program was written in 2007 fresh grad,   due to no network connection.
 So I wrote this game to kill the time.
 
   It based on Qualconn brew platform,  c++ Language


Usage:
AVK_UP : Change Shape
AVK_DOWN: Speed Down
AVK_LEFT: Left
AVK_RIGHT: Right.

AVK_SOFT2: Quit
AVK_CLR: Start/Stop
AVK_ENTER: Pause/Start
